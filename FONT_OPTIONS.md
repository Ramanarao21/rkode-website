# Font Options for RKODE LABS Logo

Replace line 19 in `Navbar.jsx` with one of these:

## Option 1: Space Grotesk (Modern, Tech-y)
```jsx
<a href="#" className="text-xl font-bold text-[#0f0d1a] tracking-tight font-['Space_Grotesk']">
```

## Option 2: Orbitron (Futuristic, Digital)
```jsx
<a href="#" className="text-xl font-bold text-[#0f0d1a] tracking-wider font-['Orbitron']">
```

## Option 3: Rajdhani (Sharp, Modern)
```jsx
<a href="#" className="text-xl font-bold text-[#0f0d1a] tracking-wide font-['Rajdhani']">
```

## Option 4: Exo 2 (Clean, Tech)
```jsx
<a href="#" className="text-xl font-bold text-[#0f0d1a] tracking-tight font-['Exo_2']">
```

## Option 5: Audiowide (Bold, Digital)
```jsx
<a href="#" className="text-xl font-bold text-[#0f0d1a] tracking-widest font-['Audiowide']">
```

## Option 6: Poppins (Clean, Professional)
```jsx
<a href="#" className="text-xl font-bold text-[#0f0d1a] tracking-tight font-['Poppins']">
```

## Option 7: Montserrat (Classic, Strong)
```jsx
<a href="#" className="text-xl font-bold text-[#0f0d1a] tracking-tight font-['Montserrat']">
```

---

To use these fonts, add this to your `index.html` in the `<head>` section:

```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Space+Grotesk:wght@700&family=Orbitron:wght@700&family=Rajdhani:wght@700&family=Exo+2:wght@700&family=Audiowide&family=Poppins:wght@700&family=Montserrat:wght@700&display=swap" rel="stylesheet">
```

My recommendations:
- **Space Grotesk** - Best for modern SaaS/tech agencies
- **Orbitron** - If you want a futuristic vibe
- **Poppins** - Safe, professional choice
